# 01 Video Breakdown Prompt

```text
Analyze the benchmark video as a TikTok ecommerce creative.

Inputs:
- Extracted frames/contact sheet: {{ }}
- Duration/resolution/fps if known: {{ }}
- User notes: {{ }}

Output a factual shot-by-shot breakdown:
- Timecode range
- Visible product/action
- Person/hand/body movement
- Camera angle and motion
- Scene/background/props
- Lighting/color/texture
- On-screen text or subtitle if visible
- Audio/voiceover cue if known
- Conversion function of the shot

Rules:
- Describe only visible or audible evidence.
- Do not invent product claims, setting details, or emotions.
- Capture short hand/product actions carefully.
- Mark uncertain details as uncertain.
```
