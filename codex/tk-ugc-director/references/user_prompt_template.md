# 用户输入模板

复制下面模板给用户填写。用户已经提供的信息不要重复追问，缺失项再问。

## 基础信息

- 产品图片：请上传 1 张清晰产品图，最好包含包装、颜色、材质、使用方式或核心配件。
- 产品名称：
- 产品类目：
- 主要卖点：
- 使用场景：
- 价格区间：
- 目标市场：美国 / 英国 / 东南亚 / 日本 / 其他
- 发布平台：TikTok / Reels / Shorts
- 视频时长：15秒 / 20秒 / 30秒
- 脚本语言：美式英语 / 英式英语 / 中文 / 日语

## UGC设定

- 达人类型：固定达人 / 不固定达人 / 只拍手部 / 不出人
- 产品露出强度：强露出 / 自然露出 / 后置露出
- 视频风格：UGC测评 / 痛点解决 / Before After / 情景短剧 / 开箱种草
- 是否需要图片提示词和视频提示词：需要 / 不需要
- 是否需要逐帧审核表：需要 Excel / 需要飞书同步 / 不需要
- 是否有品牌调性：生活感 / 精致感 / 搞笑 / 真实测评 / 极简 / 轻奢 / 亲子 / 户外 / 其他

- 不能出现的场景或人群：
- 合规敏感点：医疗 / 功效 / 儿童 / 食品 / 夸张收益 / 平台引流 / 其他

## 可直接发给用户的追问

为了把视频方案做得更像真实UGC，我还需要这几项：

1. 目标市场是哪里？
2. 准备发 TikTok、Reels 还是 Shorts？
3. 想做 15秒、20秒 还是 30秒？
4. 脚本用美式英语、英式英语、中文还是日语？
5. 达人要固定露脸、不固定达人、只拍手部，还是完全不出人？
6. 产品露出要强露出、自然露出，还是后置露出？
7. 视频风格更偏 UGC测评、痛点解决、Before After、情景短剧，还是开箱种草？
8. 需要我同时生成图片提示词和视频提示词吗？
9. 需要我导出逐帧审核表，或同步到飞书吗？
