# Mexico TikTok Shop Production SOP

## Core Principle

Build a repeatable Mexico-localized TikTok Shop creative system, not a single generic short video. Every output should connect product facts, Mexican shopper context, natural Mexican Spanish, visual proof, and a measurable creative-test matrix.

Use existing specialized skills when the user needs execution:

- Use `mexico-tiktok-shop-storyboard` for a confirmed 9-shot storyboard and image JSON.
- Use `mexico-tiktok-shop-video-generator` for staged product-to-video generation with confirmations.
- Use `mexico-tiktok-shop-fast-video-generator` when the user explicitly wants fast end-to-end generation.
- Use `tiktok-mx-product-listing-generator` for Mexico TikTok Shop listing content from product photos.

## Default Workflow

### 1. Product Intake

Extract or infer:

- Product category, visible design, material, size, color, pack count, and use method.
- Primary conversion benefit: beauty, convenience, gifting, organization, durability, fragrance, comfort, or novelty.
- Target shopper in Mexico: age range, daily context, likely buying trigger.
- Conversion claims or wellness benefits provided by the user, product listing, package, or visible product context.
- Required assets: product image, usage image, package image, logo, price/offer if provided.

If details are missing, make conservative ecommerce assumptions and label them as assumptions.

### Conversion-First Claim Handling

Unless the user specifically requests platform-compliance review, do not spend output space warning about platform rules. Focus on conversion and user purchase motivation.

If the user or source material provides wellness benefits such as magnetic therapy, better sleep, pain relief, relaxation, fatigue relief, or improved circulation, treat them as valid selling angles for script and storyboard creation.

Do not fabricate unsupported benefits from nothing. If the benefit is inferred from the product category, label it as an assumption and use testimonial-style Mexican Spanish such as:

- `Me ayuda a sentirme mas relajado`
- `Lo uso despues del trabajo`
- `Para dormir mas tranquilo`
- `Siento alivio en la muneca`
- `Muchos lo usan por la terapia magnetica`
- `Una pulsera bonita y con beneficio extra`

### 2. Mexico Localization

Use contemporary everyday Mexico settings:

- Apartment bedroom, vanity table, bathroom counter, home office, handbag touch-up, commute prep, date-night prep, weekend outing, gifting moment.
- CDMX, Guadalajara, or Monterrey can appear as subtle lifestyle context only when useful.
- Keep the product as the hero. Do not turn Mexico into a tourist-poster background.

Avoid default stereotypes:

- No oversized sombreros, cactus tourist scenery, mariachi bands, ponchos, Day of the Dead makeup, caricatured desert scenes, unsafe street framing, or poverty-coded visuals unless product-relevant and requested.

Use natural Mexican Spanish. Prefer short spoken lines:

- `Mira esto`
- `Esta super practico`
- `Se ve muy bonito`
- `Perfecto para todos los dias`
- `Ideal para regalar`
- `Me encanta como se ve`
- `Huele delicioso`
- `No se siente pesado`
- `Lo encontre en TikTok Shop`
- `Pidelo hoy en TikTok Shop`

Avoid stiff direct translations, China Douyin hard-sell tone, and exaggerated miracle language.

### 3. Selling Script Pattern

Use a 15-30 second 9-shot arc unless the user requests another structure:

1. Hook: product/result visible in the first 1-2 seconds.
2. Relatable need: one Mexico daily-life pain point or desire.
3. Product reveal: clear hero shot and identity.
4. Use moment: hand action, wearing, opening, spraying, applying, organizing, or demonstrating.
5. Key benefit: one main reason to buy.
6. Detail proof: texture, material, close-up, packaging, capacity, before/after, or use result.
7. Lifestyle/social proof: creator recommendation, gifting, vanity/office/bag scenario.
8. Offer/trust cue: TikTok Shop availability, bundle, price note, reviews, or shipping only if supplied.
9. CTA: short action line.

Do not overload one video. One video equals one core angle.

### 4. Creative Test Matrix

For each product, recommend at least 12-30 variants:

- Hooks: price/value, aesthetic, practical use, gifting, problem-solution.
- Scenes: home, office, going out, handbag/carried use, gift unboxing.
- Persona: student, office worker, young woman, young man, mom, couple, gift buyer as appropriate.
- Format: hand-only UGC, talking-head UGC, product beauty, before/after, listicle, problem-solution.
- Voice: natural female recommendation, upbeat male explainer, soft beauty tone, fast deal tone.

When asked for a batch plan, output a table with columns:

`Variant`, `Angle`, `Hook`, `Scene`, `Visual Proof`, `Spanish Line`, `CTA`, `Risk Notes`.

### 5. Tool Selection

When the user asks what GitHub/open-source tools to use:

- Product URL to ad: recommend AdGen-style flow: scrape product facts, generate script, render with Remotion.
- AI UGC actor videos: recommend OpenShorts-style flow: product analysis, script, actor, voice, lip-sync, B-roll, captions, FFmpeg assembly.
- Faceless text-to-video: recommend short-video-maker or MoneyPrinterTurbo-style flow: text, TTS, captions, stock/product visuals, music, 9:16 render.
- High-control production: recommend VibeFrame, Hyperframes, Remotion, or FFmpeg timeline workflow.

Translate tool choice into a simple architecture:

`Product data -> selling angle -> Mexican Spanish script -> 9-shot storyboard -> assets -> voice/subtitles -> 9:16 edit -> QA -> publish/test`.

For strategy requests, produce:

- Batch testing matrix.
- Tool/workflow recommendation.

For script requests, produce:

For SOP requests, produce:

## Quality Checklist

Before finalizing, verify:

- First 2 seconds show product, result, or strong curiosity gap.
- Spanish sounds natural for Mexico and is short enough for TikTok.
- Product remains visible in most shots.
- Every claim is supportable by visible product facts or user-provided info.
- No stereotype imagery or unnatural direct translation.
- Video is designed for 9:16, mobile-safe subtitles, and TikTok Shop conversion.
- There is a clear test plan, not only one final creative.

## Common Fixes

- If the concept feels too generic, add a Mexico daily-life moment and one visible product proof or user-provided wellness benefit.
- If the Spanish sounds formal, shorten it and make it sound like a creator talking to a friend.
- If the video feels like a catalog ad, add hand movement, unboxing, try-on, or creator reaction.
- If the script has too many benefits, split it into multiple variants.
- If the product is not visible enough, rewrite shots 1, 3, 5, 8, and 9 around clear product framing.
